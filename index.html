// 01 — about

Graph thinker.
Systems builder.

I'm an AI Research Engineer and PhD Computer Scientist based in Genova, Italy, with a decade of experience bridging deep theoretical research and production engineering. My work lives at the intersection of Knowledge Graphs, Graph Neural Networks, and autonomous LLM systems.

At AIGOT S.R.L., I designed DGTL — a dual-encoder Deep Graph Convolutional Network that processes Code Property Graphs (AST, CFG, DDG) for automated bug localization, trained contrastively on SWE-bench data across 8×H100 GPUs. I also built the end-to-end custom FastText 768D BPE embedding pipeline that powers it.

My PhD (University of Urbino Carlo Bo) produced a framework for profiling Smart Environments through dynamic Knowledge Graph analysis and Complex Network theory — published in IEEE Access, Elsevier, and Springer. Alongside this, I'm developing an independent research project: a Neo4j Knowledge Graph of hadith transmission chains (isnad), mapping 75M narrator records from classical Islamic scholarship using graph reasoning.

I've spoken at GraphConnect, Neo4j NODES, and international conferences across three continents.

// 02 — projects

Selected work

From production systems trained on 8×H100s to 75M-record graph databases — spanning bug localization, LLM evaluation, federated querying, and classical text analysis.

⚡

DGTL — Automated Bug Localization

Dual-encoder Deep Graph Convolutional Network (TextCNN + PyGGatedGCN) processing Code Property Graphs for file-level and zone-level bug localization. Trained contrastively on SWE-bench across 8×H100 GPUs with custom FastText 768D BPE embeddings and distractor-weighted repo-scoped ranking.

DGCN Code Property Graphs PyTorch SWE-bench 8×H100
📜

Hadith Isnad Knowledge Graph

Neo4j Knowledge Graph of Islamic hadith transmission chains (isnad), mapping 75M narrator records and their relational networks. Applies complex network analysis to model authority chains, narrator credibility clustering, and transmission path analysis using Cypher and NetworkX.

Neo4j Network Analysis 75M records Graph Reasoning
🔒

MCP Semantic Guardrails

Semantic verification and hallucination-mitigation layers for Model Context Protocol server implementations. Designed to ensure safe, schema-grounded LLM code reasoning when interacting with live graph databases. Functions as a CPG Query Service for Cursor agent mode.

MCP LLM Safety Cypher Cursor Agent
🧪

LLM Graph Query Evaluation

Comparative benchmarking of GPT-4o, Claude Sonnet 3.5, Gemini Pro 1.5, and Llama 3.1 8B for Cypher query generation using few-shot prompting with RAG-backed Chain-of-Thought reasoning. Published in Springer CSCI 2024.

GPT-4o RAG + CoT Cypher Springer 2025
🧬

DbKB — Diabetes Knowledge Graph

Multi-layer Knowledge Graph (MSNMD: Multi-Scale Network Model for Diabetes) built from 20 functional T2D genes, integrating gene expression, PPI networks, regulatory data, and biological databases. 6918 nodes, 4855 interactions across 5 relationship types. Elsevier Data in Brief, 2024.

Neo4j Systems Biology Elsevier 2024 Multi-layer KG
🌐

Federated KG Query Architecture

Decentralized querying scheme that translates SPARQL queries into Neo4j instances across distributed data sources, consolidating semantically-preserved responses from distributed KG "shards". Enables privacy-preserving multi-party graph reasoning.

SPARQL Neo4j Federated Semantic Web
📈

LLM Multi-Agent Marketing System

Agentic LLM framework for high-scale digital marketing automation, integrated with Meta Marketing API and MongoDB. Multi-agent orchestration for campaign management, content generation, and autonomous performance optimization.

Agentic LLMs Meta API FastAPI MongoDB
🎓

Bayesian Learning Pathway System

FAISS vector retrieval system paired with Bayesian Knowledge Tracing for adaptive learning pathway optimization and virtual examination. FastAPI deployment with Knowledge Tracing Extractor for student mastery modeling.

FAISS Bayesian KT FastAPI Vector DB
